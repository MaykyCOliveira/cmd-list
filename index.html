Comandos de Reparo e Otimização do Windows no cmd

Diagnóstico e Reparo do Sistema

Esses comandos ajudam a verificar e reparar problemas no Windows.

▶ Permite verificar e reparar erros no disco rígido do Windows.

chkdsk /f /r

▶ Desativar serviços problemáticos

net.exe stop "SysMain"
sc config "SysMain" start= disabled

▶ Verifica e corrige erros no disco rígido, recuperando setores defeituosos.

sfc /scannow

▶ Reparação mais profunda (DISM)

DISM /Online /Cleanup-Image /RestoreHealth

▶ Analisar processos consumindo disco

wmic process where "name='svchost.exe'" get processid,executablepath,commandline

▶ Repara a imagem do Windows, corrigindo arquivos danificados. Verifica e repara arquivos corrompidos do sistema.

DISM /Online /Cleanup-Image /RestoreHealth

▶ Analisar processos consumindo disco

wmic process where "name='svchost.exe'" get processid,executablepath,commandline

▶ Limpar arquivos temporários

del /q /f /s %temp%\*.*

▶ Desfragmentar disco (se for HDD)

defrag C: /U /V

▶ Verificar inicialização limpa

msconfig

▶ Limpar cache DNS

ipconfig /flushdns

▶ Verificar Uso da CPU, RAM e Disco (Performance Monitor)

perfmon /res

▶ Verificar Velocidade do Barramento SATA (Se o SSD será bem aproveitado)

wmic diskdrive get Name, MediaType, InterfaceType

▶ Verificar Quantidade de RAM e Uso de Swap

wmic memorychip get Capacity, Speed
systeminfo | find "Available Physical Memory"

▶ Verificar Modelo da CPU e Núcleos

wmic cpu get Name, NumberOfCores, MaxClockSpeed

▶ Desativar a atualização do timestamp do último acesso ( último horário de acesso ) para(Last Access Time) para arquivos e diretórios no NTFS.

fsutil behavior set DisableLastAccess 1

▶ Ajusta a quantidade de memória que o Windows aloca para a tabela de arquivos mestre ( MFT - Master File Table ) no sistema de arquivos NTFS .

fsutil behavior set MemoryUsage 2

▶ Gerenciamento de Memória Virtual (Swap File)

wmic pagefile list /format:list

▶ Mostra o arquivo de paginação (swap) atual. Configura manualmente o tamanho do swap (ajuste os valores conforme sua RAM). (Exemplo: 4096 MB mínimo e 8192 MB máximo para 8GB de RAM).

wmic computersystem where name="%computername%" set AutomaticManagedPagefile=False
wmic pagefileset set InitialSize=4096,MaximumSize=8192

▶ Limpeza Avançada (Temp, Cache, Thumbnails)

cleanmgr /sagerun:1

▶ Executa a limpeza de disco automática (arquivos temporários, cache, etc.).

del /s /q %windir%\temp\*.*
del /s /q %windir%\Prefetch\*.*
del /s /q "%localappdata%\Temp\*.*"

▶ Limpa pastas temporárias manualmente (mais agressivo que o cleanmgr). Reconstroi o cache de thumbnails (útil se pastas estão lentas ao abrir).

ie4uinit.exe -show

▶ Desativar Serviços Desnecessários

sc stop "DiagTrack"
sc config "DiagTrack" start= disabled

▶ Desativa o "Connected User Experiences and Telemetry" (serviço de telemetria do Windows que consome recursos).

▶ Desativa a "Windows Search" (útil se você não usa busca no sistema, mas pode tornar pesquisas mais lentas).

sc stop "WSearch"
sc config "WSearch" start= disabled

▶ Otimizar Energia (Para Máximo Desempenho)

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

▶ Analisar Processos e Inicialização. Lista processos usando +100MB de RAM (ajuste o número conforme necessário).

tasklist /svc | find "100"

▶ Verificar a versão do Windows via CMD

systeminfo | find "OS Name"

▶ Criar a pasta GodMode na área de trabalho

mkdir "%USERPROFILE%\Desktop\GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}"

▶ Verificar e Reiniciar o Spooler de Impressão (Serviço Principal) O Spooler é o serviço do Windows que gerencia trabalhos de impressão. Se ele travar, sua impressora para de funcionar. Via CMD (como Administrador)
Para que serve?
Para o serviço, limpa a fila de impressão e reinicia. Resolve travamentos, filas congestionadas e erros de "processando".

net stop spooler
del /Q /F %windir%\System32\spool\PRINTERS\*.*
net start spooler

Reinstalar Drivers Corrompidos. Drivers antigos ou corrompidos causam falhas de comunicação. Via CMD (desinstalar drivers problemáticos).
Abre a interface de remoção de drivers.
Selecione a impressora com problemas e remova.
Reinstale o driver:
Baixe o driver mais recente do site do fabricante.
Instale manualmente (Evite usar o driver genérico do Windows!).

printui /s /t2

Verificar conexão com a internet. Se houver timeouts, há problema na conexão. Pressione Ctrl + C para parar.

ping 8.8.8.8 -t

Testar DNS (problemas ao acessar sites)

nslookup google.com

Liberar e renovar IP (DHCP). Útil se você está sem internet ou com IP inválido (169.254.x.x).

ipconfig /release
ipconfig /renew

Limpar cache DNS. Resolve erros como "Site não encontrado" mesmo com internet.

ipconfig /flushdns
Resetar pilha TCP/IP (problemas graves de conexão)

netsh int ip reset netsh winsock reset

Reinicia configurações de rede (reinicie o PC depois).

Desativar Limitação de Banda do Windows

gpedit.msc

Vá para:
Configuração do Computador > Modelos Administrativos > Rede > Agendador de Pacotes QoS
Habilite "Limitar banda reservável" e defina como 0%.
(Se não tiver gpedit.msc, use este comando alternativo:)

reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\Psched" /v "NonBestEffortLimit" /t REG_DWORD /d 0 /f

Priorizar tráfego (para jogos e chamadas). Reduz latência, mas pode diminuir velocidade em downloads.

netsh int tcp set global autotuninglevel=restricted

Desativar economia de energia no adaptador Wi-Fi. Evita quedas de conexão Wi-Fi.

powercfg /setdcvalueindex SCHEME_CURRENT 19cbb8fa-5279-450e-9fac-8a3d5fedd0c1 12bbebe6-58d6-4636-95bb-3217ef867c1a 0
powercfg /setactive SCHEME_CURRENT

Listar todas as conexões de rede. Verifique se o adaptador está "Conectado".

netsh interface show interface

Verificar processos usando internet. Identifique programas consumindo banda (use Task Manager para fechar).

netstat -ano | findstr ESTABLISHED

Desativar IPv6 (se causar conflitos)

netsh interface ipv6 set global state=disabled

Teste de Velocidade

curl --max-time 10 -o NUL "http://speedtest.london.linode.com/100MB-london.bin"
